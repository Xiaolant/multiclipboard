# MultiClipboard Python Project

The MultiClipboard Python project is a versatile and user-friendly clipboard manager designed to simplify your workflow by enabling you to store and access multiple pieces of text or data using unique key-value pairs. With this project, you can kiss repetitive copying and pasting goodbye, and effortlessly manage your clipboard contents.

## Introduction

Have you ever found yourself repeatedly copying and pasting the same text snippets while working on various projects? With the MultiClipboard Python project, you can overcome this tedious task by conveniently saving your frequently used text or data to the clipboard with unique keys. This open-source clipboard manager is built to enhance your productivity and improve your development experience.

## Features

- **Multiple Entries:** Store and manage an unlimited number of text or data entries with ease.
- **Key-Value Access:** Each stored entry is assigned a unique key, allowing quick and easy retrieval.
- **User-Friendly Interface:** The project provides a simple and intuitive command-line interface for smooth interaction.

## Usage

Using the MultiClipboard Python project is straightforward. Here are the main commands you can use:

- `python multiclipboard save`: Add a new entry to the clipboard with the specified key and value.
- `python multiclipboard load`: Retrieve the stored value associated with the given key.
- `python multiclipboard delete`: Delete the entry with the specified key from the clipboard.
- `python multiclipboard save`: Display all the stored keys in the clipboard.

## Examples

Let's explore some examples to see how the MultiClipboard Python project can simplify your daily tasks:

1. Adding an Entry:
   ```
   $ python multiclipboard.py save
   Enter a key:1
   ```

2. Retrieving an Entry:
   ```
   $ python multiclipboard.py load
   Enter a key associated with data:1
   ```

3. Deleting an Entry:
   ```
   $ python multiclipboard.py delete
   Enter the key of item to delete:
   ```

4. Listing All Keys:
   ```
   $ python multiclipboard.py list
   ```
